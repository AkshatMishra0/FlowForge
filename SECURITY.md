## Security Policy

### Reporting a Vulnerability

If you discover a security vulnerability in FlowForge, please send an email to security@flowforge.com. 

Please include the following information:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge your email within 48 hours and send a more detailed response within 96 hours.

### Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

### Security Best Practices

When using FlowForge:
- Always use environment variables for sensitive data
- Keep dependencies updated
- Enable 2FA on your account
- Use strong passwords
- Regularly rotate API keys
